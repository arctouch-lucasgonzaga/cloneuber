# Clone Uber

This template should help get you started developing with Vite.

## Project Setup

```sh
npm install
```

### Start local dev server.

```sh
npm run dev
```

### Locally preview the production build.

```sh
npm run preview
```

### Build for production.

```sh
npm run build
```
